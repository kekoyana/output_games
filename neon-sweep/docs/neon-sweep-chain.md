# Neon Sweep チェイン仕様書

## 基本ルール：サンドイッチ消去

パネルを配置したとき、**同じ色のパネル2つで別の色を挟む**と消去が発生する。

```
例: 置く前  [赤] [青] [_]   ← プレイヤーがここに赤を置く
    置いた後 [赤] [青] [赤]  ← 青が挟まれて消去！
```

### 消去対象
- 挟まれたパネル（間の異なる色）**すべて**
- 挟む側のパネル（両端の同色）**も消える**
- 置いたパネル自身**も消える**

```
例: [赤] [青] [青] [緑] [赤]  ← 赤を置く
     ↑    ↑    ↑    ↑    ↑
    消去  消去  消去  消去  消去  ← 5個すべて消える
```

### 方向
- 横・縦・斜め（計8方向/4軸）でサンドイッチ判定
- 1回の配置で複数方向の消去が同時に発生しうる

---

## 消去演出

- 消去されたパネルは **2秒間** かけてフェードアウトする（`CLEAR_ANIM_TIME = 2.0`）
- フェードアウト中もパネルの**色とシンボルは見える**（徐々に透明＋縮小）
- フェードアウトが完了するとパネルは盤面から消滅する

---

## チェインの仕組み

### 核心：パネル単位のチェインレベル

消去中の各パネルは **chainLevel**（何チェイン目で消えたか）を記憶している。

### チェイン判定

```
新しいサンドイッチに消去中パネルが含まれている？
  ├─ YES → chainCount = そのパネルの chainLevel の最大値 + 1（チェイン継続）
  └─ NO  → chainCount = 1（新規消去）
```

### 重要：チェインは消去中パネルの chainLevel に紐づく

- チェインはグローバルな状態ではなく、**個々の消去中パネルに記録される**
- 新しいサンドイッチが消去中パネルを「含む」（挟む端として使う or 間に含む）場合にのみチェインが継続
- 消去中パネルを含まないサンドイッチは、別の場所で消去が進行中でも chainCount = 1

---

## チェインの具体例

### 基本チェイン

```
時刻0.0s: 赤で挟んで青を消去 → chainCount=1
          消去された3パネルに chainLevel=1 が記録される

時刻0.5s: 消去中パネル(chainLevel=1)を端にして別の色を挟む
          → chainCount = 1 + 1 = 2（2チェイン！）
          新たに消去されたパネルに chainLevel=2 が記録される
```

### 複数チェインラインの合流

```
場所A: 5チェインのパネルがフェードアウト中（chainLevel=5）
場所B: 別の場所で3チェインのパネルがフェードアウト中（chainLevel=3）

→ 場所Aの消去中パネルを使ってサンドイッチ
  → chainCount = max(5) + 1 = 6（6チェイン！）

→ 場所Bの消去中パネルを使ってサンドイッチ
  → chainCount = max(3) + 1 = 4（4チェイン）
```

### チェインの中断と再接続

```
時刻0.0s: サンドイッチ → chainCount=1、パネルA群に chainLevel=1
時刻0.3s: サンドイッチ（消去中パネルを含まない）→ chainCount=1（リセット）
          ※ パネルA群の chainLevel=1 はそのまま残っている
時刻0.8s: パネルA群を使ってサンドイッチ → chainCount = 1 + 1 = 2（復活！）
```

**途中で別の消去が入っても、消去中パネルの chainLevel は消えない。**
**そのパネルがフェードアウト完了するまでチェインを繋げるチャンスがある。**

## スコア計算

| アクション | スコア |
|-----------|--------|
| 通常消去（1パネルあたり） | +10 |
| チェインボーナス（2チェイン以降） | +消去数 × 10 × チェイン数 |

```
例: 3パネル消去 × 3チェイン = 基本30 + ボーナス90 = 120点
```

## 消去中パネルの特殊な扱い

### サンドイッチの端として使える
消去中（フェードアウト中）のパネルは、同色であればサンドイッチの**端（endpoint）**として機能する。

```
例: [赤(消去中,chainLevel=2)] [青] [_]  ← ここに赤を置く
    → 消去中の赤が端として有効 → 青が消去 → chainCount = 2+1 = 3
```

### 消去中パネルがサンドイッチに含まれた場合
- そのパネルのフェードアウトタイマーが**リセット**される（再び2秒に戻る）
- chainLevel が現在のチェイン数に**更新**される
- 消去数のカウントには**含まない**（既にカウント済みのため）

### 消去中パネルの上に置ける
- フェードアウト中のマスは空きマスと同様に**パネルを置ける**
- 置くと消去中パネルは上書きされる

## 実装詳細

### Panel.chainLevel
- `0` = 消去されていない通常パネル
- `1` = 1チェイン（初回消去）で消えた
- `N` = Nチェイン目で消えた

### checkSandwich() の戻り値
- `totalCleared`: 新たに消去開始したパネル数
- `maxInvolvedChainLevel`: サンドイッチに関与した消去中パネルの最大chainLevel（0=なし）

### チェイン判定フロー
1. パネル配置
2. `checkSandwich()` 実行
3. `maxInvolvedChainLevel > 0` → `chainCount = maxInvolvedChainLevel + 1`
4. `maxInvolvedChainLevel == 0` → `chainCount = 1`
5. 今回消去されたパネルに `chainLevel = chainCount` を記録
